Konfigurasi Lokal¶

Setelah Anda menginstall Qiskit, ada beberapa konfigurasi opsional yang dapat Anda ambil untuk mengubah perilaku default dari Qiskit khusus untuk penggunaan spesifik Anda.

User Config File¶

Lokasi utama untuk konfigurasi lokal dari Qiskit adalah user config file. Ini adalah file format ` ini <https://en.wikipedia.org/wiki/INI_file>` __ yang dapat digunakan untuk mengubah konfigurasi default Qiskit.

Sebagai contoh:

[default]
circuit_drawer = mpl
circuit_mpl_style = default
circuit_mpl_style_path = ~:~/.qiskit
state_drawer = hinton
transpile_optimization_level = 3
parallel = False
num_processes = 15

Secara default file ini tinggal di ` `~/.qiskit/settings.conf ` ` tapi path yang digunakan dapat diganti dengan variabel ` ` QISKIT_SETTINGS ` ` ` variabel lingkungan. Jika ` ` ` QISKIT_SETTINGS ` ` ` sudah ditetapkan, nilainya akan digunakan sebagai path ke berkas konfigurasi pengguna.

Pilihan yang tersedia:

  • ` ` ` circuit_drawer ` : Ini digunakan untuk mengubah backend default bagi penggambar sirkuit :meth: qiskit.circuit.QuantumCircuit.draw ` dan :func:` qiskit.visualization.circuit_drawer `. Hal ini dapat diatur ke ` ` latex ` `, ` ` mpl ` `, ` ` text ` `, atau ` ` latex_source ` ` dan ketika ` ` ouptut ` ` kwarg tidak secara eksplisit diatur, maka backend drawer yang akan digunakan.

  • ` ` ` circuit_mpl_style ` : Ini adalah lembar style default yang digunakan untuk ` ` ` mpl ` ` output backend untuk penggambar sirkuit :meth: qiskit.circuit.QuantumCircuit.draw ` dan :func:` qiskit.visualization.circuit_laci `. Hal ini dapat diatur ke ` ` default ` ` ` atau ` ` bw ` `.

  • circuit_mpl_style_path: This can be used to set the path(s) to have the circuit drawer, qiskit.circuit.QuantumCircuit.draw() or qiskit.visualization.circuit_drawer(), use to look for json style sheets when using the mpl output mode.

  • state_drawer: This is used to change the default backend for the state visualization draw methods qiskit.quantum_info.Statevector.draw() and qiskit.quantum_info.DensityMatrix.draw(). It can be set to repr, text', latex, latex_source, qsphere, hinton, or bloch bloch and when the output kwarg is not explicitly set on the draw() method that output method will be used.

  • ` ` transpile_optimization_level ` ` : Ini mengambil bilangan bulat antara 0-3 dan digunakan untuk mengubah level optimisasi default untuk :func: ~qiskit.compiler.transpile ` dan :func:` ~qiskit.execute.execute `.

  • parallel: This option takes a boolean value (either True or False) and is used to configure whether Python multiprocessing is enabled for operations that support running in parallel (for example transpilation of multiple QuantumCircuit objects). The default setting in the user config file can be overriden by the QISKIT_PARALLEL environment variable.

  • num_processes: This option takes an integer value (> 0) that is used to specify the maximum number of parallel processes to launch for parallel operations if parallel execution is enabled. The default setting in the user config file can be overriden by the QISKIT_NUM_PROCS environment variable.

Variabel lingkungan¶

Ada juga beberapa variabel lingkungan yang dapat diatur untuk mengubah perilaku default dari Qiskit.

  • QISKIT_PARALLEL: if this variable is set to TRUE it will enable the use of Python multiprocessing to parallelize certain operations (for example transpilation over multiple circuits) in Qiskit Terra.

  • QISKIT_NUM_PROCS: Specifies the maximum number of parallel processes to launch for parallel operations if parallel execution is enabled. It takes an integer > 0 as the expected value.

  • ` ` ` QISKIT_IBMQ_PROVIDER_LOG_LEVEL ` `: Spesifikasi level log yang akan digunakan, untuk modul ` ` qiskit-ibmq-provider ` `. Jika tingkat yang tidak valid ditetapkan, tingkat log kembali ke default WARNING. Level log yang valid adalah ` ` DEBUG ` `, ` ` INFO ` `, ` ` WARNING ` `, ` ` ERROR ` `, dan ` ` CRITICAL ` ` (huruf kapital tidak penting). Jika variabel lingkungan tidak diatur, maka tingkat logger induk yang digunakan, yang juga memiliki default ` ` WARNING ` `.

  • ` ` ` QISKIT_IBMQ_PROVIDER_LOG_FILE ` `: Spesifikasi nama dari berkas log untuk digunakan dari pesan log yang berasal dari ` ` qiskit-ibmq-provider ` `. Jika dispesifikasikan, pesan akan masuk ke berkas saja. Jika tidak, pesan akan login ke kesalahan standar (biasanya layar).

  • ` ` ` QISKIT_AQUA_MAX_GATES_PER_JOB ` `: Sebuah parameter opsional untuk mengatur ambang batas untuk membagi sirkuit yang dihasilkan Aqua ke dalam beberapa pekerjaan yang diserahkan ke sebuah backend berdasarkan jumlah gates.
